Why choose us?

The core of your customer support

Having a central location for customers requests is key for a help desk. It allows you to have a complete picture of each customer, so you can offer the right support.

1

Adaptable to your business

Peppermint is a flexible solution that can fit your business needs at any stage of growth.

2

Host anywhere

Where it be on the cloud, an on prem server or even a raspberry pi. Peppermint has no bounds to where it is hosted.

3

Open Source

Peppermint is an open source project at its' heart. Removing any cost and mystery from the product. This also allows for community PR's and extra eyes on the project.

Multiple deployment options

Peppermint is built to be hosted in a docker container, which can be easily scaled through k8's. In addition to this you can host peppermint through pm2 or even locally with NodeJS
